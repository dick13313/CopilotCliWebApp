🚀 快速啟動
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. 系統檢查:
     ./check-system.sh     (Linux/Mac)
     check-system.bat      (Windows)

  2. 啟動應用:
     ./start.sh            (Linux/Mac)
     start.bat             (Windows)

  3. 開啟瀏覽器:
     http://localhost:5173

📊 API 端點
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  POST   /api/chat/session         建立會話
  POST   /api/chat/send            發送訊息
  GET    /api/chat/sessions        取得會話列表
  DELETE /api/chat/session/{id}    刪除會話

  後端: http://localhost:5000
  前端: http://localhost:5173

🐛 已知問題與解決方案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  問題 1: "The requested model is not supported"
  解決:  使用正確格式 claude-sonnet-4.5（小寫+破折號）

  問題 2: "Failed to start Copilot CLI client"
  解決:  確認 Copilot CLI 已安裝且已登入 GitHub

  問題 3: CORS 錯誤
  解決:  確認後端正在運行且 CORS 設定正確

  ➡️ 詳細解決方案請見 TROUBLESHOOTING.md
